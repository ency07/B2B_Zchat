# PROHIBIDO

Nunca generar UI solamente usando Tailwind.

Nunca improvisar colores.

Nunca improvisar espaciados.

Nunca improvisar radios.

Nunca improvisar iconografía.

Nunca improvisar sombras.

Nunca improvisar jerarquías.

Nunca improvisar tipografía.

Toda decisión debe venir del Blueprint o de UI UX Pro Max.

---

# DEFINICIÓN DE TERMINADO

Ningún componente puede darse por terminado hasta demostrar:

1. Qué recomendó UI UX Pro Max.

2. Qué decidió el Blueprint.

3. Qué terminó implementándose.

Si falta cualquiera de esos tres puntos, el trabajo NO está terminado.
