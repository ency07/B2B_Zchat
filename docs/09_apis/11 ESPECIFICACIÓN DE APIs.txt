Endpoint

Método

Autenticación

Errores

Ejemplo:

POST /api/clients

Request:
{
  company_name,
  tax_id,
  city,
  email
}

Validaciones:
- company_name obligatorio
- tax_id único

Auditoría:
CLIENT_CREATED

Permisos:
CLIENT_CREATE

Pruebas:
CLIENT-001
CLIENT-002
