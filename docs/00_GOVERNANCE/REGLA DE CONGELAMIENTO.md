# REGLA DE CONGELAMIENTO

Una fase tiene 4 estados:

DISCOVERY
DESIGN
BUILD
CLOSED

DISCOVERY:
Se permiten preguntas.

DESIGN:
No se permiten nuevas preguntas.

BUILD:
No se permiten nuevas preguntas.

CLOSED:
Solo correcciones de bugs.

Si durante BUILD aparece una pregunta:

Es un fallo de análisis.

Generar reporte.

No preguntar al usuario directamente.
