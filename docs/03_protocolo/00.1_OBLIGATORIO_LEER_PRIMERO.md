1. 00_CONSTITUCION_TECNICA_CYH_OS.md
2. MASTER_INDEX.txt
3. PROJECT_MEMORY.md
4. IMPLEMENTATION_ORDER.txt
5. Toda la documentación relacionada con la fase a implementar.

También deberá ejecutar obligatoriamente un REUSE_ANALYSIS antes de escribir código nuevo.
