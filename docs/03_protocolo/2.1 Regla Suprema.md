NUNCA ejecutar una batería global de pruebas.

NUNCA ejecutar tests de módulos no modificados.

Cada fase debe tener su propio script de validación independiente.

Si se modifica un módulo:

1. Ejecutar únicamente la validación del módulo afectado.
2. Ejecutar únicamente validaciones de dependencias directas.
3. Prohibido ejecutar pruebas completas del sistema.
4. Prohibido refactorizar módulos no relacionados.
5. Si una corrección impacta otro módulo, detenerse y reportarlo antes de continuar.

Cada fase se considera terminada únicamente cuando:

* Migraciones ejecutan correctamente.
* Script de validación pasa.
* No existen errores TypeScript.
* No existen errores de lint del módulo.
* Se genera informe de validación.
