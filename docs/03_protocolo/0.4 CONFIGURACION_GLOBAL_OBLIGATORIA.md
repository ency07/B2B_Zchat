Toda información susceptible de cambiar por empresa, usuario o entorno (logos, colores, nombres, teléfonos, correos, destinatarios, claves de integración, URLs, branding, plantillas, preferencias y proveedores) debe almacenarse en el Centro de Configuración. Está prohibido codificar estos valores directamente en el código fuente. Si durante una fase se detecta un valor configurable fuera de este módulo, la tarea falla y debe corregirse antes de continuar.
