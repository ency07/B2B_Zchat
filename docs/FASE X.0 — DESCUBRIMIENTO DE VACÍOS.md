# PROTOCOLO DE PREGUNTAS DE NEGOCIO

Antes de iniciar cualquier fase:

1. Leer toda la documentación relacionada.

2. Detectar TODOS los vacíos documentales.

3. Generar una única lista consolidada de preguntas.

4. Clasificar cada pregunta:

* Crítica
* Importante
* Opcional

5. Entregar TODAS las preguntas en una sola respuesta.

6. Confirmar explícitamente:

"Después de responder estas preguntas no existen más vacíos funcionales conocidos para esta fase."

7. Esperar respuestas.

8. Congelar decisiones.

9. Actualizar documentación.

10. Generar diseño.

11. Solicitar aprobación.

12. Construir.

13. Prohibido realizar nuevas preguntas de negocio durante la implementación.

Si aparece una nueva pregunta:

Detener.

Generar:

INCONSISTENCY_REPORT.md

Indicando:

* Documento faltante
* Regla faltante
* Impacto
* Pregunta exacta
