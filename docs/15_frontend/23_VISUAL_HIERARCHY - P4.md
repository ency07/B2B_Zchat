# PARTE 4

# Componentes, Prioridades Visuales y Densidad de Información

---

# PILAR 131

CADA COMPONENTE DEBE TENER UN PROPÓSITO EMPRESARIAL

Está prohibido agregar componentes únicamente porque "se ven modernos".

Todo componente debe:

• ahorrar tiempo
• mostrar información
• reducir errores
• facilitar decisiones

---

# PILAR 132

LOS COMPONENTES MÁS IMPORTANTES DEBEN OCUPAR LAS ZONAS MÁS IMPORTANTES

Nunca colocar un KPI crítico debajo de componentes secundarios.

¿Cuántos proyectos atrasados?
